Многостраничный интернет-магазин по продаже кальянных товаров.
* Реализована корзина товаров. Вход в корзину возможен только после регистрации.
* Реализованы функции авторизации и регистрации пользователей. Выполнены по методологии ООП (Папка Classes).
* Настроен ЧПУ сайта.
* Данные товаров и зарегестрированных пользователей берутся из json файла.
